A simple Python-based number guessing game created for learning Git and GitHub workflows.

## About This Project

This repository was created as a practice project for learning GitHub fundamentals, including repository initialization, commits, and push operations. The game itself is a fun, interactive command-line application where players try to guess a randomly generated number.

## How to Play

The game will think of a random number between 1 and 100, and your goal is to guess it in as few attempts as possible. After each guess, you'll receive hints to help you narrow down the correct number.

## Features

- Random number generation between 1 and 100
- Input validation to ensure valid guesses
- Helpful hints (too high/too low) after each guess
- Attempt counter to track your performance
- Clean, user-friendly interface with emoji indicators

## Prerequisites

- Python 3.x installed on your system

## Installation

1. Clone this repository:
```bash
git clone https://github.com/yourusername/number-guessing-game.git
```

2. Navigate to the project directory:
```bash
cd number-guessing-game
```

## Usage

Run the game using Python:

```bash
python number_guessing_game.py
```

Follow the on-screen prompts to play the game!

## Learning Objectives

This project demonstrates:
- Basic Git operations (init, add, commit, push)
- GitHub repository management
- Python fundamentals (functions, loops, conditionals, error handling)
